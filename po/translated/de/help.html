USB Creator Hilfe

Beschreibung

Der USB Creator ist ein Werkzeug zur Herstellung eines startfähigen Flash-Laufwerks mit mehreren Linux-Betriebssystemen.

Wichtig

Alle wichtigen Daten auf dem Flash-Laufwerk müssen gesichert werden, bevor Sie beginnen.

USB Creator erwartet ein USB-Stick mit einer einzelnen fat32-Partition oder einer 100MB fat32-Bootpartition und einer zweiten Partition für die ISOs im Dateisystemformat Ihrer Wahl.

Hinweis: Eine fat32-Partition kann keine Dateien mit mehr als 4 GB aufnehmen.

Terminal

Sie können den USB Creator auch über das Terminal verwenden.

Mit "usb-creator -h" eine Liste aller verfügbaren Optionen anzeigen lassen.

Startet Ihr ISO nicht?

Wenn die ISO nicht mit automatischer Erkennung des Distributionsnamens gebootet hat, wählen Sie den entsprechenden Distributionsnamen manuell mit Hilfe des Dropdown-Menüs "Manuell" aus.

Wenn Sie Ihr ISO immer noch nicht booten können, können Sie das Menü auf Ihrem Flash-Laufwerk unter /boot/grub/grub/grub.cfg konfigurieren.

Bitte helfen Sie mit, diese Anwendung zu verbessern, indem Sie grub.cfg bis zum Hochfahren der ISO anpassen und Ihre Änderungen auf GitHub oder unserem Forum veröffentlichen.

Vergessen Sie nicht, die Logdatei anzuhängen, wenn Sie ein Problem in unserem Forum veröffentlichen: ~/.usb-creator/usb-creator.log
