USB Creator Help

Descrizione

USB Creator è uno strumento per facilitare la creazione di dispositivi avviabili con più sistemi operativi Linux.

Importante

Assicurarsi di aver salvato tutti i dati importanti dalla periferica perima di incominciare.

USB Creator si aspetta una chiavetta USB con una singola partizione fat32 o una partizione di boot fat32 di 100MB e una seconda partizione per l'ISO nel formato di filesystem di tua scelta.

Attenzione: una partizione fat32 non può contenere file più grandi di 4GB.

Terminale

Puoi usare USB Creator anche da terminale.

guarda "man usb-creator" per un elenco delle opzioni disponibili.

L'immagine ISO non si avvia?

Se l'ISO non esegue il boot con il rilevamento automatico della distribuzione, seleziona manualmente il nome appropriato della distribuzione dal menù a discesa "Manuale"

Se ancora non puoi eseguire il boot della tua ISO puoi configurare il menù del tuo drive flash: /boot/grub/grub.cfg
